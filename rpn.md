# Reverse Polish Notation API - calculator

Here you can see all available operators and variables which you can use to write your expressions.

## Operators

### Basics

- Add : `+`
- Subtract : `-`
- Multiply : `*`
- Divide : `/`


### Advanced

- Modulus : `%`
- Power : `^`
- Square Root : `sqrt`
- Exponentiation : `exp`
- Factorial : `!` (⚠️ only on integer)
- Natural Logarithm : `ln`
- Logarithm : `log`

### Trigonometric

- Sine : `sin`
- Arc Sine : `asin`
- Cosine : `cos`
- Arc Cosine : `acos`
- Tangent : `tan`
- Arc Tangent : `atan`

## Variables

- PI : `pi`
- Eular's number : `e`
